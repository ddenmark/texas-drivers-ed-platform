# Statutory Required Topics Integration Plan

**Source:** Texas Education Code, Chapter 1001, §§102–1101 (as referenced in TDLR Curriculum Standards for Driver Education Providers)

These 15 topics are **mandatory** for all driver education curricula in Texas. They must be explicitly included with appropriate instructional time, materials, and assessment. Failure to cover them is a compliance violation.

Our platform will treat these as non-optional "Required Content Gates" with dedicated tracking for TDLR audits.

**Total Dedicated Statutory Time:** Approximately 2.5–3.5 hours across the course (counted toward the overall 32-hour requirement). This is in addition to overlapping coverage in core defensive driving, risk reduction, and consumer topics.

## Platform Implementation Requirements (for Course Builder)

1. **Required Topics Dashboard** in admin: Checklist of all 15 topics with status (Not Started / In Progress / Completed).
2. **Auto-generated Content Blocks**: When building a course, these topics have pre-approved templates (video placeholders + official resource links + quiz banks).
3. **Time Tracking**: Each statutory topic has a configurable minimum engagement time. Completion only counts after timer + assessment pass.
4. **Audit Export**: Separate CSV/JSON report showing coverage and student completion of each statutory topic for TDLR submission.
5. **Content Sourcing**: Prioritize official sources (TxDOT, TDI, DPS, TDLR) for accuracy and defensibility during audits.
